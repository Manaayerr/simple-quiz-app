## 🇺🇸 English

A simple, responsive, and interactive Quiz Application built using HTML, CSS, and vanilla JavaScript. This project showcases dynamic DOM manipulation, state management, and basic UI transition effects.

### ✨ Features
* **Dynamic Questions:** Questions and answers are rendered dynamically using JavaScript.
* **Instant Feedback:** Buttons change color immediately to indicate correct (green) or incorrect (red) answers.
* **Smart UI States:** The "Next" button only appears after an answer is selected to guide the user naturally.
* **Final Score Summary:** Shows the user's total score at the end with an option to replay instantly.
* **Fully Responsive:** Sleek, modern design that adapts beautifully to mobile, tablet, and desktop screens.

### 🛠️ Technologies Used
* **HTML5:** Semantic structure for the layout.
* **CSS3:** Custom styles, hover effects, and responsive design utilizing Google Fonts (Poppins).
* **JavaScript (ES6):** Dynamic content rendering, event handling, and score tracking.
